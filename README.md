# Unit Converter

A simple CLI unit converter for length, weight, temperature, area, and speed.

## Usage

### Length
- Meters to Feet: `meters_to_feet(meters)`
- Feet to Meters: `feet_to_meters(feet)`

### Weight
- Kilograms to Pounds: `kg_to_lbs(kg)`
- Pounds to Kilograms: `lbs_to_kg(lbs)`

### Temperature
- Celsius to Fahrenheit: `celsius_to_fahrenheit(celsius)`
- Fahrenheit to Celsius: `fahrenheit_to_celsius(fahrenheit)`

### Area
- Acres to Square Feet: `area_acres_to_sqft(acres)`
- Square Feet to Acres: `area_sqft_to_acres(sqft)`

### Speed
- Miles per Hour to Kilometers per Hour: `speed_mph_to_kph(mph)`
- Kilometers per Hour to Miles per Hour: `speed_kph_to_mph(kph)`

## Example
```python
print(meters_to_feet(1))  # Output: 3.28084
print(kg_to_lbs(1))  # Output: 2.20462
```
